- 하나의 문제를 푸는 여러 개의 알고리즘이 존재한다고 할 때, 어떤 알고리즘이 더 좋은지(효율적인지)에 대해서 `과학적` 이고 `수치적` 으로 표현할 수 있기 위해서 점근적 분석이 등장했다.

## Weak Ordering

- `leading term` 의 차수가 같고 계수가 다를 때는 계수부분은 더 빠른 컴퓨터로 커버할 수 있기 때문에 계수로는 평가하지 않는다.
